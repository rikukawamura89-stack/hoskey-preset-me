Q: Kenapa LED NumLock "Paten" nyala terus?
A: Ini fitur 'Anti-Conflict'. Biar shortcut Numpad (7,4,1) gak bentrok sama
   fitur Windows Snap (pindah-pindah jendela). Jadi LED itu tanda script
   lagi jagain keyboard lu biar tetep fokus buat Media Control!

Q: Gimana cara matiin script-nya?
A: Cari icon [H] hijau di Taskbar (pojok kanan bawah), Klik Kanan > Exit.
   LED NumLock bakal bisa dimatiin normal lagi setelah script mati.
